# Welcome to our Project
# 🧠 Mental Health Detector

A modern, cloud-based web application designed to **detect anxiety and sadness**, monitor emotional well-being, and provide actionable wellness insights through a sleek and user-friendly dashboard.

The system uses structured questionnaires, mood tracking, and rule-based scoring to help users understand their mental health trends and take preventive action.

---

## 🚀 Project Overview

**Mental Health Detector** is a responsive web platform that:

- Tracks daily emotions and habits
- Detects anxiety and sadness levels
- Generates a wellness score (0–100)
- Provides personalized coping suggestions
- Displays real-time mental health analytics
- Ensures secure and scalable cloud deployment

---

## 🎯 Key Objectives

- Early detection of anxiety and sadness
- Continuous emotional monitoring
- Simple and explainable scoring system
- User-friendly healthcare-style interface
- Secure data handling and storage

---

## ✨ Core Features

### 1. User Authentication
- Secure login and signup
- Session management
- Encrypted user data

### 2. Mood Tracking System
Users can log their daily mood:

- Happy
- Neutral
- Sad
- Anxious
- Stressed

### 3. Anxiety & Sadness Detection Questionnaire
Structured mental health assessment including:

- Nervousness or excessive worry
- Feeling sad or hopeless
- Sleep difficulties
- Low energy
- Stress levels

System calculates severity levels:

- Low
- Moderate
- High

### 4. Mental Health Score Engine

Generates:

- Wellness Score (0–100)
- Risk classification
  - Healthy
  - Mild
  - Moderate
  - Severe

### 5. Dashboard

Displays:

- Wellness score
- Mood trends
- Risk indicators
- Alerts
- Recent activity

### 6. Emotion Trend Visualization

Charts include:

- Weekly mood trends
- Anxiety level trends
- Sadness level trends
- Emotional history

### 7. Journal System

Users can:

- Write daily thoughts
- Save entries
- Review past logs

### 8. Habit Tracker

Tracks:

- Sleep hours
- Water intake
- Exercise
- Screen time

### 9. Rule-Based Alerts

Triggers alerts when:

- Sad mood persists for multiple days
- Anxiety score is high
- Sleep falls below threshold

### 10. Personalized Suggestions

Provides:

- Breathing exercises
- Meditation guidance
- Stress management tips
- Relaxation techniques

### 11. Emergency Detection System

If risk is high:

- Warning message displayed
- Support resources suggested
- Crisis guidance provided

### 12. User Profile Management

Stores:

- Age
- Sleep patterns
- Stress history
- Preferences

### 13. Notification System

Sends:

- Daily reminders
- Wellness alerts
- Progress updates

### 14. Secure Cloud Storage

- Encrypted database
- Real-time data synchronization
- Reliable data persistence

### 15. Responsive Modern UI

Design includes:

- Clean minimal layout
- Smooth animations
- Mobile-friendly interface
- Dark mode support
